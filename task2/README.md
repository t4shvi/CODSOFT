# Sushi Restaurant Landing Page

**Web Development Internship – CodSoft**  
**Task 2 | February 2026**

This is a landing page for a sushi restaurant created using **HTML and CSS** as part of the CodSoft Web Development Internship. The project focuses on designing a visually appealing, beginner-friendly landing page while practicing layout, alignment, and color palette.

## Sections
- Hero (Headline, Subheadline, Image)  
- Features / Why You’ll Love It
- Lead Capture / Notify Me Form  
- Testimonials / Social Proof  
- Footer (Address & Copyright)  

## Technologies Used
- HTML5  
- CSS3  

## Objective
To practice front-end layout, styling, and user interface design while creating an engaging landing page experience.
